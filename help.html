Make the most of Image and Manga mode

This help center lives inside the extension, so you can review onboarding, permissions, and download flows without leaving your workflow. Everything respects your selected theme.

Quick start checklist

Follow these steps whenever you install Unshackle on a new profile.

Choose a mode

Pick Image or Manga at the top of the panel to load the right scanners and UI.

  • Image mode focuses on DOM, canvas, and background captures.
  • Manga mode unlocks the HakuNeko panel plus bookmarks.

Review permissions

Unshackle only asks for site access when you run scans. Use the Advanced menu to reset prompts.

  • Allow access for the active tab when prompted.
  • Revoke via the lock icon in Chrome if needed.

Pick a theme

All Unshackle surfaces—including this page—sync your theme choice across popup, bookmarks, and help.

  • Open the ⚙ button in the panel header.
  • Select Contrast, Dark, or Light-in-the-Dark.

Manga mode essentials

Delegate runners, cookies, and loader fallbacks keep downloads reliable.

Loader selection

Auto mode switches between Runner and Manager. Force a loader from Settings > Manga > Loader.

  • Runner favors compatibility with more sites.
  • Manager reuses delegates injected into the page for speed.

Cookies & auth

Capture login cookies with the Manga cookie UI before scanning paywalled chapters.

  • HTTPS only—your data stays on-device.
  • Each request shows host, label, and last-updated time.

Bookmarks & updates

Save detected series from the Manga panel and revisit them via the Bookmark tab from the context menu.

  • Bookmarks store connector ID, URL, and family.
  • The “Recent updates” drawer fetches new chapters on demand.

Use these shortcuts for faster extractions.

Advanced overlay tools

Open the Advanced menu to trigger CSS background scans, rename jobs, and launch viewer-specific helpers.

Canvas capture

Use the Canvas button for same-origin drawing buffers. Combine with page bridge fetches for protected viewers.

Download status

The footer shows current job progress with cancel controls, mirroring notifications from the service worker.
